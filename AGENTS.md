# Project Instructions

## Startup

- At the start of a new session, read `SESSION_NOTES.md` first before making changes or answering project-specific questions.
- Use `SESSION_NOTES.md` as the current handoff and project-state summary.
- Treat `## Manual Notes > ### Next Session Prompt` as the highest-priority startup prompt for the session.
- Also check `## Manual Notes > ### Prompt History` for recent prompt continuity when the latest prompt is short or ambiguous.

## Notes

- `SESSION_NOTES.md` is updated automatically when the Unity editor quits and when scenes are saved.
- Preserve the `## Manual Notes` section in `SESSION_NOTES.md`.
